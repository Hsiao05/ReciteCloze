# 背诵完形 · ReciteCloze

一个**单文件网页应用**，用 DeepSeek 辅助的填空式遮盖帮助背诵长文本——课文、名词解释、法律条文、外语文章等。

只需粘贴原文，AI 自动找出最值得挖空的考点，交互式地逐层遮盖、逐词击破。



## 核心特性

### AI 辅助标注
- 接入 **DeepSeek API**，自动识别文本中被挖空概率最高的信息点：年份/数字、专有名词、术语、理论、因果链、对比关系等
- 内置**命题规则提示**，让 AI 像出题老师一样思考
- 也可选择**离线隔字模式**，按交替规律自动遮盖，无需 API

### 三层分级体系
| 层级 | 说明 | 典型内容 |
|------|------|----------|
| **一层核心** `⟬…⟭` | 高概率考点，最像填空题答案 | 数字、专名、术语、结论关键词 |
| **二层扩展** `⟪…⟫` | 次高概率考点，辅助回忆句意 | 限定条件、分类项、配对项、修饰语 |
| **结构保护** `⟦…⟧` | 保持原文可读性 | 标点、列表符号、逻辑连接词 |

### 交互式背诵体验
- **拖拽遮盖/显示**：手指或鼠标划过词块，即时隐藏或揭示
- **长按标重点**：长按 450ms 将词块标记为重点
- **五种遮盖模式**：原文 → 一层 → 二层 → 重点 → 全遮，渐进式
- **字号调节**：15–36px 滑动调节，适应不同阅读场景
- **全显/重置**：一键恢复，随时重新开始

### 视觉设计
- 浅色/深色双主题，一键切换
- 毛玻璃质感 
- 适配电脑、平板、手机
- 固定顶栏切换 + 悬浮底栏模式选择

### 本地卡片管理
- 自动将当前语料与标注缓存为「卡片」
- 每条卡片记录**上次背诵时间**
- 支持**导出/导入 JSON 文件**，分享卡片库
- 数据完全本地存储（`localStorage`），无需后端

### 多语言支持
- 智能分词：中文 (Intl.Segmenter)、日文 (假名/汉字)、英文 (拉丁字母+专名合并) 等
- 跨语言统一的考点识别逻辑
- 语言均衡分配遮盖比例，避免集中在单一语言

## 快速开始

### 直接使用

1. **打开** [GitHub Pages](https://hsiao05.github.io/ReciteCloze/)
2. 粘贴要背诵的文本
3. 选择「自定义 DS API」，填入自己的 DeepSeek API Key，点击「生成标注语料」或「隔字生成」。**所有操作都在本地运行，不会在网络上泄露 API Key**
4. 切换到「背诵」页 → 开始交互练习

### 本地使用

1. **下载** [`index.html`](index.html)
2. `index.html` 中
  > const DEVELOPER_API_KEY = "";

  填入自己的 DeepSeek API Key。不建议在网上公开包含自己 Key 的文件，以免造成一些不良后果。

3. **双击用浏览器打开**
2. 粘贴要背诵的文本 → 选择「开发者的 API」，点击「生成标注语料」或「隔字生成」
3. 切换到「背诵」页 → 开始交互练习

## 🎬 工作流演示

```
┌─────────────┐     ┌──────────────┐     ┌──────────────┐
│  粘贴原文    │ ──▶ │ AI 标注/隔字  │ ──▶ │  交互背诵     │
│  (配置页)   │     │  (配置页)     │     │  (背诵页)     │
└─────────────┘     └──────────────┘     └──────────────┘
                           │                      │
                           ▼                      ▼
                    ┌──────────────┐     ┌──────────────┐
                    │  可手动编辑   │     │ 缓存为卡片    │
                    │  标注结果     │     │ 导出/导入     │
                    └──────────────┘     └──────────────┘
```

1. **粘贴原文**：支持任何语言的文本，自动清理 Markdown 格式
2. **生成标注**：AI 返回 `core`/`broad`/`priority` 标注，本地后处理做比例均衡和位置分散
3. **校验结果**：自动检查一层/二层比例是否合理，词元数是否合适
4. **进入背诵**：五种模式渐进练习，拖拽即时遮盖/揭示
5. **导出卡片**：JSON 格式保存，可分享、可复现

## 操作

| 操作 | 方式 |
|------|------|
| 切换页面 | 点击顶部「卡片/配置/背诵」 |
| 遮盖/显示词块 | 拖拽划过词块（按压后滑动） |
| 标为重点 | 长按词块 (≥450ms)，继续滑动批量标注 |
| 取消重点 | 长按已重点词块 |
| 切换遮盖层级 | 点击底部「原文/一层/二层/重点/全遮」 |
| 全部显示 | 点击「全显」按钮 |
| 恢复当前层级 | 点击「重置」按钮 |
| 调整字号 | 拖动背诵页字号滑块 |
| 切换深色/浅色模式 | 点击 ☀️ 图标 |
| 导出卡片 | 卡片页 → 导出文件 |
| 导入卡片 | 卡片页 → 导入文件 |

## 📝 自定义标注语法

如果你想手动编写标注，协议如下：

| 符号 | 含义 | 效果 |
|------|------|------|
| `⟦ … ⟧` | 结构保护 | 永不被遮盖 |
| `⟬ … ⟭` | 一层核心考点 | 「一层」模式时遮盖 |
| `⟪ … ⟫` | 二层扩展考点 | 「二层」模式时遮盖 |
| `\|` | 最小语言单位分隔 | 在一段标注内进一步细分词块 |

示例：
```
⟦1. ⟧⟬马克思主义⟭是⟪关于自然界、人类社会和思维发展⟫⟬一般规律⟭的⟬科学理论体系⟭。
```
